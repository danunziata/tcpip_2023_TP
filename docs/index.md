# Smart Green House

## Resumen

El proyecto del Invernadero Inteligente es una implementación innovadora que utiliza tecnología IoT (Internet of Things) para mejorar la producción agrícola a través del control preciso del entorno de cultivo en invernaderos. El objetivo principal es optimizar las condiciones ambientales para lograr una producción de calidad y maximizar la eficiencia de los recursos.

La industria agrícola enfrenta desafíos significativos debido a la dificultad de realizar pronósticos claros y a la dependencia de la intuición de los expertos. Para superar estas limitaciones, nuestro equipo ha sido contratado por una importante empresa del sector AgTech para implementar un Invernadero Inteligente utilizando tecnología IoT.

El Invernadero Inteligente está diseñado para monitorear y controlar de cerca parámetros críticos como la temperatura, humedad, intensidad de luz y dióxido de carbono. Estos datos se adquieren a través de sensores inalámbricos y de una API de pronóstico del tiempo.

Una vez adquiridos los datos, se utilizó un programa en Python para procesarlos y enviarlos a un servidor en la nube a través del protocolo MQTT. El servidor, implementado en la plataforma ThingsBoard, proporciona una interfaz gráfica personalizada para visualizar y controlar los datos del invernadero, así como una base de datos para almacenar la información recopilada.

Además, se utilizó la herramienta Ngrok para establecer una conexión segura y remota entre la Raspberry Pi, que actúa como computadora central, y el servidor en la nube. Esto permite el monitoreo y control del Invernadero Inteligente desde cualquier ubicación.

El proyecto del Invernadero Inteligente representa un avance significativo en la agricultura de precisión, brindando una solución tecnológica que optimiza la producción agrícola y supera las limitaciones de la intuición humana. Aunque no se pudo implementar la parte de hardware con sensores reales de dióxido de carbono, se utilizó una API de pronóstico del tiempo para simular esta información y obtener una visión completa del entorno del invernadero.
